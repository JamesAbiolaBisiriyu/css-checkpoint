# Learning CSS Layout Project

This project is a frontend practice project built with **HTML and CSS**.  
It focuses on mastering modern layout techniques such as **Flexbox**, spacing, alignment, buttons, cards, sections, and responsive structure.

The design is inspired by a medical/healthcare landing page and was created to improve real-world UI implementation skills.
